## 1. 接口描述
接口请求域名: /blog/deleteBlog
删除博客
请求方式: GET

## 2. 输入参数
| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
| id | String | 博客id |

## 3. 输出参数
| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
|   stateCode   | String  |   状态码 1是成功   |
| message  | String | 描述信息 |

## 4. 示例

### 示例1 查询结果列表

#### 输入示例

/blog/deleteBlog?id="333"


#### 输出示例

{
    stateCode: "1",
    message: "删除成功",
}
