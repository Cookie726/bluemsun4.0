## 1. 接口描述
接口请求域名: /news/addNews
增加作品
请求方式: POST

## 2. 输入参数
| 参数名称  | 必选  |  类型  |         描述         |
| :-------: | :---: | :----: | :------------------: |
|  title  |  是   | String | 标题 |
|  newsClass | 是 | 'news' 'notice'| 类型 |
| content | 是 | String | 内容 |

## 3. 输出参数
| 参数名称 |  类型  |        描述        |
| :------: | :----: | :----------------: |
|   stateCode   | String  |   状态码 1是成功   |
| message  | String | 描述信息 |
| id | String | 新闻id |

## 4. 示例

### 示例1 查询结果列表

#### 输入示例

news/addNews

{
    title: '蓝旭新版本',
    newsClass: 'news',
    content: '<p>完美</p>'
}

#### 输出示例

{
    stateCode: "1",
    message: "发布成功",
    id: "32"
}
